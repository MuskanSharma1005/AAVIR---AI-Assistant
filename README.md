# AAVIR : AI-Assistant
AAVIR – AI Voice Assistant AAVIR (Artificial Voice Intelligent Responder) is a Python based AI voice assistant that performs everyday tasks through voice commands. It integrates speech recognition, text to speech, web automation, system control, and OCR capabilities to create a hands-free computing experience.

Features
1. Voice Interaction
Speech-to-text using SpeechRecognition
Text-to-speech responses using pyttsx3
Continuous voice command listening
2. Information Retrieval
Wikipedia search summaries
Google search automation
Latest news headlines via RSS feeds
3. Web Automation
Open Gmail
Open YouTube
Play YouTube videos using voice commands
Perform Google searches automatically
4. System Controls (MacOS)
Shutdown system
Restart system
Lock screen
5. Utility Functions
Tell current time and date
Check battery status
Find files on the system
Voice typing into any text field
6. Productivity Tools
Task Manager
Add tasks
Show tasks
Meeting Notes Recorder
Records voice notes
Saves them into a text file
7. Accessibility Tools
Screen Reader
Captures the screen
Extracts text using OCR (Tesseract)
Reads the text aloud

Technologies Used
Python
SpeechRecognition
Pyttsx3 (Text-to-Speech)
Wikipedia API
Google Search
Webbrowser Automation
Feedparser (News RSS)
Pytesseract (OCR)
PIL (Image Processing)
Pynput (Keyboard Automation)
Subprocess (System Commands)
